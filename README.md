## How to View Diagrams

### Option 1: Markdown Viewers with Mermaid Support
- GitHub (automatically renders Mermaid)
- GitLab
- VS Code with Mermaid extension
- Markdown viewers with Mermaid plugin

### Option 2: Online Mermaid Editors
1. Copy Mermaid code from any diagram file
2. Paste into [Mermaid Live Editor](https://mermaid.live/)
3. View and export as PNG/SVG

### Option 3: VS Code Extension
Install "Markdown Preview Mermaid Support" extension in VS Code
